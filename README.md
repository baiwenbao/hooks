## Getting Started

![](<./39b699d9be734f99a41aee8cf87fbf6a%20(2).png>)

Install

```bash
$ npm i @baiwenbao/hooks
```

Install dependencies,

```bash
$ npm i
```

Start the dev server,

```bash
$ npm start
```

Build documentation,

```bash
$ npm run docs:build
```
